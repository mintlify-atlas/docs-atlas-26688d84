    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Zalbi Theme in just a few steps.
  </p>

  <Steps>
    <Step title="Download and Install">
      Clone the repository or download the theme files, then upload to your WordPress installation's `wp-content/themes/` directory.

      ```bash
      cd wp-content/themes/
      git clone https://github.com/Garridoparrayeray/zalbi-theme.git
      ```
    </Step>

    <Step title="Install Dependencies">
      Install the required Node.js and Composer dependencies to enable development tools.

      ```bash
      cd zalbi-theme
      composer install
      npm install
      ```
    </Step>

    <Step title="Activate the Theme">
      Navigate to **Appearance** > **Themes** in your WordPress admin dashboard and activate the Zalbi Theme.
    </Step>

    <Step title="Configure Custom Post Types">
      The theme automatically registers two custom post types: Hinchables (inflatables) and Eventos (events). Start creating content immediately.
    </Step>
  </Steps>
