## What is Zalbi Theme?

Zalbi Theme is a custom WordPress theme specifically designed for **Zalbi Aisia eta Abenturs**, a business specializing in inflatable rentals and event management. Built on the solid foundation of Underscores (_s), this theme provides a complete solution for showcasing rental products and managing event bookings.

<Info>
  Zalbi Theme is based on [Underscores](https://underscores.me/), the popular WordPress starter theme by Automattic, ensuring clean, well-structured code following WordPress best practices.
</Info>

## Who is this for?

This theme is ideal for:

- **Inflatable rental businesses** looking to showcase their catalog online
- **Event management companies** needing to display services and collect inquiries
- **Bilingual businesses** operating in Spanish and Basque (Euskera) markets
- **WordPress developers** who want to customize or extend a rental/events theme

## Key Features

### Custom Post Types

Zalbi Theme includes two powerful custom post types:

<CardGroup cols={2}>
  <Card title="Hinchables (Inflatables)" icon="balloon">
    Manage your inflatable catalog with custom fields for dimensions, capacity, and categories.
  </Card>
  <Card title="Eventos (Events)" icon="calendar">
    Showcase event services with details like duration and target audience.
  </Card>
</CardGroup>

### Multilingual Support

<Note>
  The theme is fully prepared for bilingual content using **Polylang**, supporting both Spanish and Basque (Euskera) languages.
</Note>

- Automatic language detection and content switching
- Translated UI elements and navigation
- Language-specific URLs and menus

### Advanced Custom Fields (ACF)

The theme leverages Advanced Custom Fields for:

- Product specifications (dimensions, capacity)
- Event details (duration, target audience)
- Visual categorization with color-coded tags
- Custom metadata for each post type

### Built-in Features

- **Custom Catalog Template** - Filterable product display with taxonomy-based categories
- **WhatsApp Integration** - Floating WhatsApp button configurable via WordPress Customizer
- **Responsive Design** - Mobile-first approach for optimal viewing on all devices
- **SEO-Friendly URLs** - Clean permalinks for both post types
- **Custom Navigation Menus** - Primary menu and footer legal menu support
